## Descripción

Sitio web institucional para **ConekTIC**, una empresa de servicios de tecnología de la información especializada en transformación digital, soluciones tecnológicas personalizadas e integración de procesos para organizaciones.

El proyecto incluye secciones de presentación corporativa, propuesta de valor, servicios y formulario de contacto.

## Demo

🔗 **[Ver sitio en vivo](https://cintava.github.io/ConekTIC/public/)**

## Tecnologías

- **HTML5** — Estructura semántica
- **CSS3** — Estilos y diseño responsive
- **SASS (SCSS)** — Preprocesador CSS con variables, mixins y nesting
- **Node.js** — Gestión de dependencias y compilación de SASS

## Estructura del proyecto

```
ConekTIC/
├── public/          # Archivos compilados y assets
├── scss/            # Archivos fuente SASS
├── package.json     # Dependencias del proyecto
└── README.md
```

## Instalación local

```bash
git clone https://github.com/cintava/ConekTIC.git
cd ConekTIC
npm install
```

## Características

- Diseño responsive adaptado a distintos dispositivos
- Arquitectura de estilos modular con SASS
- Formulario de contacto
- Secciones de servicios con tarjetas visuales
